# DOM属性

DOM 将文档表示为节点和对象；这样，编程语言就可以与页面交互。

## DOM树

```html
<html lang="en">
  <head>
    <title>My Document</title>
  </head>
  <body>
    <h1>Header</h1>
    <p>Paragraph</p>
  </body>
</html>
```
