# 数字分隔符

从 **ES2021（ES12）** 开始，JS/TS 允许在**数字字面量中使用 `_` 作为分隔符**，只为了 **提高可读性**。

```ts
10_000 === 10000
1_000_000 === 1000000
0b1010_0001
0xff_ff
```

JS / TS 对数字分隔符 `_` **没有强制分组规则**，只要位置合法就行：

❌ 字符串里
❌ 变量名里
❌ 不能放在数字开头、结尾、连续使用

```ts
_1000      // ❌
1000_      // ❌
10__000    // ❌

```

主流做法：千分位：

```ts
1_000_000
```
