# Tabs

## Tabs Props



### value

| 类型            | 默认值 | 必传 |
| --------------- | ------ | ---- |
| String / Number | -      | 否   |

激活的选项卡值。支持语法糖 `v-model` 或 `v-model:value`

### **defaultValue**

| 类型            | 默认值 | 必传 |
| --------------- | ------ | ---- |
| String / Number | -      | 否   |

激活的选项卡值。非受控属性。



## TabPanel Props

### **value**

| 类型            | 默认值 | 必传 |
| --------------- | ------ | ---- |
| String / Number | -      | 否   |

选项卡的值，唯一标识。

### removable

| 类型    | 默认值 | 必传 |
| ------- | ------ | ---- |
| Boolean | false  | 否   |

当前选项卡是否允许移除
