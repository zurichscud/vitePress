# Spawned actor

`spawn`译为独立的

spawned actor 是 被“创建出来长期运行”的 actor，不参与状态的进入 / 退出生命周期



## 核心区别

| 维度          | invoke              | spawn              |
| ------------- | ------------------- | ------------------ |
| 创建时机      | 进入 state 自动创建 | 任意 action 中创建 |
| 销毁时机      | 离开 state 自动停止 | ❌ 不会自动销毁     |
| 是否绑定状态  | ✅ 是                | ❌ 否               |
| 是否有 onDone | ✅ 有（若完成）      | ❌ 没有             |
| 是否有 output | 可能有              | ❌ 不关心           |
| 通信方式      | 自动 done / error   | 手动 `send`        |
| 适合场景      | 请求、一次性任务    | 长期服务、监听     |
