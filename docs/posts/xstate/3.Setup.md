# Setup

setup是v5的核心升级

## what

`setup` 用来集中声明“实现细节”，`createMachine` 只写“状态逻辑”

```ts
const machine = setup({
  actions: { ... },
  guards: { ... },
  actors: { ... },
}).createMachine({ ... })

```

## why 

**状态机结构是“纯描述”**

**actions / guards 是“可替换实现”**

```ts
machine.provide({
  actions: {
    doSomething: () => console.log('mock')
  }
})
```
