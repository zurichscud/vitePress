# 内置注解



## @Override

- 表示方法重写
- 编译器会帮你检查是否真的重写成功



## @Deprecated

标记方法/类已过时



## @SuppressWarnings

抑制编译器警告

```java
@SuppressWarnings("unchecked")
```

| **参数**            | **说明**                                                     |
| ------------------- | ------------------------------------------------------------ |
| **`"unused"`**      | 屏蔽“定义的变量/方法从未被使用”的警告。                      |
| **`"unchecked"`**   | 屏蔽“未经检查的转换”警告（常出现在泛型和原始类型转换时）。   |
| **`"deprecation"`** | 屏蔽“使用了已过时（@Deprecated）的方法或类”的警告。          |
| **`"rawtypes"`**    | 屏蔽“使用原生类型而没有指定泛型”的警告（如 `List` 而非 `List<String>`）。 |
| **`"all"`**         | 屏蔽所有类型的警告（**不推荐**，除非你在处理极其特殊的情况）。 |
| **`"serial"`**      | 屏蔽实现 `Serializable` 接口但没定义 `serialVersionUID` 的警告。 |
