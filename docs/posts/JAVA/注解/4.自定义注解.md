# 自定义注解

## 向属性传值

在使用注解时，可以给它的属性（方法）赋值

```java
@Log(value = "新增用户", module = "用户模块")
public void addUser() {}
```

如果不需要向注解中的属性传值也可以省略`()`

```java
@Log
public void addUser() {}
```



## 属性本质

注解里的属性，本质是 **接口方法**。注解属性 ≈ 无参方法

```java
public @interface MyAnnotation {
    String value();
}
```

等价于：

```java
public interface MyAnnotation extends Annotation {
    String value();
}
```

- 属性定义方式

```java
public @interface UserInfo {

    String name();              // 必填
    int age() default 18;       // 有默认值
    String[] tags() default {}; // 数组
}
```

## 属性类型限制

1. 基本类型

```java
int age();
boolean flag();
```

2. String

```java
String name();
```

3. Class

```java
Class<?> clazz();
```

4. 枚举

```java
Status status();
```

5. 上面类型的数组

```java
String[] tags();
int[] nums();
```

## 默认值

表示这个属性可以不传

```java
int age() default 18;
```



## value特殊规则

使用时 **只传一个属性**，并且这个属性是`value`，那么可以简写。

```java
String value();
```

```java
public @interface Log {
    String value();
    String module() default "";
}
```

完整写法：

```java
@Log(value = "新增用户")
```

简写：

```java
@Log("新增用户") 
```



## 数组属性

```java
public @interface Roles {
    String[] value();
}
```

- 单个值，可以省略`{}`

```java
@Roles("ADMIN")
```

- 多个值，不能省略

```java
@Roles({"ADMIN", "USER"})
```
