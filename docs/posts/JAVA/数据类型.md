# 数据类型

## 数据默认值

| 数据类型     | 默认值                     |
| ------------ | -------------------------- |
| `byte`       | 0                          |
| `short`      | 0                          |
| `int`        | 0                          |
| `long`       | 0L                         |
| `float`      | 0.0f                       |
| `double`     | 0.0d                       |
| `char`       | `\u0000`（Unicode 空字符） |
| `boolean`    | false                      |
| 引用数据类型 | null                       |
