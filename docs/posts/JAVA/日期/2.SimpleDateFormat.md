# SimpleDateFormat

`SimpleDateFormat` 是 Java 中用于**日期格式化和解析**的工具类

```java
java.text.SimpleDateFormat
```



## Constructors



```java
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
```

| 格式 | 含义   | 示例 |
| ---- | ------ | ---- |
| yyyy | 年     | 2026 |
| MM   | 月     | 03   |
| dd   | 日     | 24   |
| HH   | 24小时 | 15   |
| mm   | 分钟   | 30   |
| ss   | 秒     | 00   |



## Methods



### format

格式化：Date → 字符串

```java
Date date = new Date();

SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
String str = sdf.format(date);

System.out.println(str);
```

### parse

转换：字符串 → Date

```java
String str = "2026-03-24 15:30:00";

SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
Date date = sdf.parse(str);
```
