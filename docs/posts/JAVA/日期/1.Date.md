# Date

`Date` 本质上是：一个 **毫秒值（long）**，表示从 **1970-01-01 00:00:00 GMT** 到当前时间的偏移量

```java
java.util.Date
```



## Constructors

```java
Date now = new Date(); // 当前时间
```



```java
Date date = new Date(1672531200000L); // 指定时间戳
```



## Methods



### public long getTime()

获取时间戳

```java
Date date = new Date();
long timestamp = date.getTime();

System.out.println(timestamp);
```



### toString()

```java
System.out.println(now.toString());
```

```java
Tue Mar 24 10:00:00 GMT 2026
```



## Tip

- Date 已经过时
- Date 不支持格式化，需要使用`SimpleDateFormat`
