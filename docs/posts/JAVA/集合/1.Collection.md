## toArray

将集合转为数组

### Object[] toArray()

```java
 Object[] toArray()
```

```java
List<String> list = List.of("A", "B", "C");

Object[] arr = list.toArray();
```

需要手动强制类型转换

### T[] toArray(T[] a)

```java
<T> T[] toArray(T[] a);
```

参数a不是用来“传数据”，而是：**告诉集合：我要什么类型的数组**。即传入泛型，省去了类型转换

```java
List<String> list = List.of("A", "B", "C");

String[] arr = list.toArray(new String[0]);
```



## containsAll

```java
boolean containsAll(Collection<?> c);
```

取两个集合的交集。保留的是当前集合中的元素。

- `true`：当前集合发生了变化（有元素被删除）

- `false`：当前集合没有变化
