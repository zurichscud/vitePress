# Set

Java 中的 `Set` 是 `Collection` 体系下的一个重要分支，**特点是：不允许元素重复**。

## Set特点

- 不允许重复元素（通过 `equals()` 判断）

- 无索引，不能通过下标访问

- 一般不保证顺序（部分实现类除外）

## 实现类

### HashSet

- 底层使用`HashMap`
- **无序**
- *不重复**

通过 `hashCode()` + `equals()` 判断是否重复

```java
Set<String> set = new HashSet<>();
set.add("A");
set.add("A"); // 不会重复
```



### LinkedHashSet

是 `HashSet` 的子类

- **有序（插入顺序）**
- 不重复
- 性能略低于 `HashSet`

```java
Set<String> set = new LinkedHashSet<>();
set.add("B");
set.add("A");

System.out.println(set); // [B, A]
```



### TreeSet

- **自动排序（默认升序）**

- 不重复

- 不允许 `null`

通过`compareTo` / `Comparator`进行排序

```java
Set<Integer> set = new TreeSet<>();
set.add(3);
set.add(1);
set.add(2);

System.out.println(set); // [1, 2, 3]
```
