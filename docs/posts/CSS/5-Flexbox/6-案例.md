## 多行可伸缩布局

需求：

- 每一行 flex item数量为6个

- item的宽度具备响应式。

- flex item之间存在间隙16px
- 版心需要对齐

````html [淘宝解决方案]
<!DOCTYPE html>
<html lang="en">

        margin: 0;
        padding: 0;
                /* 关键：padding属于width */
        box-sizing: border-box;
      }

        width: 80%;
        margin: 0 auto;
      }
      .container {
        display: flex;
        margin-left: -8px;
        margin-right: -8px;
        flex-wrap: wrap;
      }
      .item {
        height: 200px;
        flex: 0 0 16.666667%;
        padding: 0 8px;
        margin-bottom: 16px;
      }
      .box {
        height: 100%;
        background-color: pink;
      }
      .footer {
        width: 80%;
        height: 100px;
        margin: 0 auto;
        background-color: #333;
      }
    </style>
  </head>
  <body>

    </div>
````
