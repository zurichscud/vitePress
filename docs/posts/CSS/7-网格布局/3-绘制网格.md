# 绘制网格

在grid布局中，绘制网格非常灵活

## 重复绘制网格结构

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 100px); /* 定义3列，每列宽度100px */
  grid-template-rows: repeat(2, 100px); /* 定义2行，每行高度为100px */
}
```



## 列轨道自动填充

`auto-fill`是一个特殊的关键字，意味着自动填充尽可能多的列或行，直到容器的宽度或高度用完为止。

```css
.container {
  display: grid;
  /*auto-fill会根据容器的尺寸自动计算出能容纳多少个 100px 的列。*/
  grid-template-columns: repeat(auto-fill, 100px);
}
```

::: danger

`auto-fill`  只能用于  **`grid-template-columns`**  ，**不能用于 `grid-template-rows`**。这是 CSS Grid 的固有设计限制。

:::

## 指定比例

```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr
  grid-template-rows:1fr 2fr 1fr
}
```



## 控制栅格波动范围

```css
.container {
  display: grid;
  grid-template-columns: repeat(2, minmax(50px,100px)); 
  grid-template-rows: repeat(2, 1fr); /* 定义2行，每行高度相等 */
}
```

如果容器的可用空间足够大，列的宽度可以增长到 100px；如果空间不够，列的宽度会缩小到 50px。
