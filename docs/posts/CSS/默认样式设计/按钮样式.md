

# 按钮样式

## 基本样式

```css
      button {
        cursor: pointer;
        border: none;
        outline: none;
        /* 以下可以根据需要自定义 */
        border-radius: 5px;
        height: 30px;
        width: 80px;
        background: #010e1c;
        color: #fff;
      }
```





## hover

颜色较正常的按钮颜色会浅一点

```css
      button:hover {
        background: #1b2b3c;
      }
```



## disabled

颜色会较hover更浅一点

```css
      button:disabled {
        background: #ccd4da;
        cursor: not-allowed;
      }
```
