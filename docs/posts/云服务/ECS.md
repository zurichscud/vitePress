# ECS

云服务器本身，相当于一台虚拟机。

```
ECS实例
 ├── 系统盘
 │    └── 快照（系统盘备份）
 │         └── 镜像（可用于创建新实例）
 └── 数据盘
      └── 快照（数据备份，可挂载新实例使用）
```



## 云盘

ECS实例挂载的磁盘，包括系统盘和数据盘。

## 快照

云盘在某个时间点的完整备份，具体可分为系统盘快照，数据盘快照。快照因为需要占用云服务商存储空间，因此需要按量计费



## 镜像

镜像是可创建新 ECS 的模板。

**创建流程**：

1. 先对系统盘做快照。

2. 从快照生成镜像。

3. 用镜像快速创建新的 ECS 实例。

数据盘快照一般不会直接生成镜像，但可以挂载新实例作为数据盘。
