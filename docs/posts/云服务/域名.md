# 域名

## 顶级域名

**顶级域名**：包括常见的 `.com`、`.org`、`.net`、`.cn` 等。TLD 是一个域名的最上层，它表示该网站的类型、目的地或所属国家/地区。

## 主域名

在域名 `www.juejin.cn` 中：

- `cn` 是 **顶级域名**，表示该域名属于中国。
- 主域名：`juejin.cn`
- 子域名`www`



## 子域名

子域名是主域名的扩展部分，位于主域名的左侧。子域名通常用于将网站的不同部分分开，或者为不同的服务提供独立的访问路径。子域能帮助你根据业务需求管理多个服务或功能。

子域名是 `主域名` 左边的部分，通常用 `.` 分隔。例如：

- `www.juejin.cn`：`www` 是 **子域名**，`juejin.cn` 是 **主域**。
- `api.juejin.cn`：`api` 是 **子域名**，`juejin.cn` 是 **主域**。

**不同的业务模块**：例如，`www` 用于网站主页，`blog` 用于博客，`api` 用于 API 服务等。

- `www.example.com` — 网站的主页面
- `blog.example.com` — 公司博客
- `shop.example.com` — 电商平台
- `api.example.com` — 提供 API 服务

一个网站可以没有子域名：如果网站本身不需要划分不同的功能或服务，就不需要额外的子域名，直接使用主域名 `example.com` 即可。

- 过去，网站地址一般都会带有 `www` 子域名（例如 `www.example.com`）。然而，随着时间的推移，很多网站选择直接使用 `example.com` 来简化网址并提高用户体验。

- 这种做法并不影响访问，因为服务器可以被配置成同时接受 `www.example.com` 和 `example.com`，并将它们重定向到同一个站点。



## 域名注册

云服务商提供域名注册的功能。

- 注册域名时，需要先选择顶级域名，顶级域名不同，价格也不同。

- 注册域名时选择绑定的服务器IP地址即可

- **实名认证**：个人需身份证，企业需营业执照（国内域名强制要求）

- 添加域名解析：不同的域名可以绑定到不同的IP

## DNS

域名与IP配置完成后，需要等待DNS服务器传播。大约5-30分钟，DNS即可全球同步

```sh
zurichscud@zurichscuddeMacBook-Air ~ % nslookup 你的域名
```

如果出现如下结果，说明DNS已配置成功

```
Server:         192.168.3.1
Address:        192.168.3.1#53

Non-authoritative answer:
Name:   zurichscud.site
Address: 124.221.210.219
```
