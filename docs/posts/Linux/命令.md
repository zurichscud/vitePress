# 命令

一个命令就是一个Linux程序，命令没有图形化页面

Linux命令的通用格式：

```sh
command [-options] [parameter]
```

一个命令可以同时指定多个options：

```sh
ls -a -l
ls -al
ls -la
```

以上三种方式完全等价



## HOME目录

在 **Linux / Unix（包括 macOS）** 中，**home 目录**指的是**用户的主目录（用户自己的工作目录）**。home 目录是每个用户登录系统后默认进入的目录

每个用户都会有自己的目录：

```sh
/home/用户名
```

例如：

```sh
/home/zhangsan
/home/lisi
```

root用户的home目录位于

```sh
/root
```

在shell中，`~`表示 **当前用户的 home 目录**。

```sh
cd ~
cd /home/username
```

## 工作目录

Linux命令行执行命令的时候，需要一个工作目录，打开终端时，默认设置工作目录在用户的HOME目录

## ls

> list

查看目录内容（列出文件和文件夹）

```sh
ls [-a -l -h] [Linux路径]
```

- Linux路径：可选，要查看的目录路径

| 部分 | 全称           | 含义                                                 |
| ---- | -------------- | ---------------------------------------------------- |
| `-a` | all            | 显示所有文件（包括隐藏文件）                         |
| `-l` | long           | 以详细列表形式显示                                   |
| `-h` | human-readable | 让文件大小更易读（KB / MB / GB），必须和`-l`一起使用 |

```sh
ls -a
ls -lh
```

`ls -l`得到的文件大小的格式默认是B，添加`-h`，让文件大小以人类更易读的方式展示



## 特殊路径符

- `.`：当前目录
- `..`：上一级目录
- `~`：HOME目录

```sh
cd ./Desktop
cd ../..
cd ~/Desktop
```

## cd

> change directory

```sh
cd [Linux路径]
```

- Linux路径：目标工作目录

```sh
#快速返回HOME目录
cd
#快速返回上一级
cd ..
```





## pwd

> print work directory

查看当前的工作目录

```sh
pwd
```
