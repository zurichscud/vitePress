# 应用

## systemctl

控制Linux内置服务和第三方服务的命令。`systemctl`需要`root`权限。

::: tip

第三方服务需要被systemctl管理，需要主动集成入`systemctl`。我们也可以手动集成入`systemctl`中

:::

- 查看状态

```sh
sudo systemctl status 服务名
```

- 启动服务

```sh
sudo systemctl start 服务名
```

- 停止服务

```sh
sudo systemctl stop 服务名
```

- 重启服务

```sh
sudo systemctl restart 服务名
```

- 开机启动

```sh
sudo systemctl enable 服务名
```

- 取消开机启动

```sh
sudo systemctl disable 服务名
```
