# 表现型属性

CSS 的优先级高于 HTML 属性（如 `bgcolor`、`border`、`align` 等）

```html
<table bgcolor="red"></table>
```

现代浏览器会：把这些属性 **转换成一条很低优先级的 CSS 规则**

```css
table[bgcolor] {
  background-color: red;
}
```

因此作者样式会覆盖表现型属性

```css
table {
  background-color: blue;
}
```

不要再用 `bgcolor / align / border` 控制样式， 全部用 CSS（哪怕是行内 `style` 也比这些强）
