# 图片格式

## jpg

- **有损压缩**

- 不支持透明

- 体积小，兼容性最好

## png

- **无损压缩**

- 支持透明

- 体积比 JPG 大

## bmp

- **位图格式**
- 通常 **不压缩 / 压缩率极低**
- 文件体积非常大

::: tip

前端中几乎不会使用此格式

:::

## gif

- 支持简单动画

- 256 色限制

- 支持透明（不是真透明）

## webp

- 有损 / 无损

- 支持透明、动画

- 体积比 JPG / PNG 小很多

::: tip

webp是现代浏览器最推荐的图片格式，具有上述所有格式的优点

:::

## base64

把二进制数据用字符串表示的一种编码方式

```html
<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..." />
```

img根据src中的前缀可以指定当前图片是base64。
