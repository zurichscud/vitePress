# 全局属性

## id

**唯一标识**一个元素（页面中不能重复）

```html
<a href="#app">跳转</a>
<div id="app"></div>
```



## class

```html
<div class="card active"></div>
```



## style

```html
<p style="color:red;font-size:14px;">文本</p>
```



## dir

文本方向，取值：

- `ltr`（默认）
- `rtl`
- `auto`

```html
<p dir="rtl">从右往左</p>
```

该属性具有继承性，因此通常只在html元素上使用：

```html
<html dir="ltr">
```



## title

- 鼠标悬停会出现文字提示

- 对可访问性也有帮助

```html
<span title="这是提示">Hover me</span>
```



## lang

```html
<p lang="zh-CN">中文</p>
```

该属性具有继承性，因此通常只在html元素上使用：

```html
<html lang="zh-CN">
```
