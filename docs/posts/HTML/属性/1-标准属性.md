# 标准属性

HTML元素本身拥有的属性

所有的标准属性均可通过`dom.属性名`得到。标准属性始终都是存在的，不管你是否有在元素中声明该属性



## 布尔属性

在 **HTML 中，布尔属性（Boolean Attributes）** 指的是：**只要属性值存在，就表示 `true`；不存在就是 `false`**，不关心具体值。

```html
<input disabled>
<input disabled="disabled">
<input disabled="">
```

以上三种写法效果完全一致，表示：disabled = true

属性不存在即表示false：

```html
<input>
```

```html
<input disabled="false">
<!--依然是 true，因为属性存在。-->
```



常见的 HTML 布尔属性：

```ts
disabled      <!-- 禁用 -->
readonly      <!-- 只读 -->
required      <!-- 必填 -->
checked       <!-- 选中（checkbox / radio） -->
selected      <!-- 选中（option） -->
multiple      <!-- 多选 -->
autofocus     <!-- 自动聚焦 -->
novalidate   <!-- 禁用表单校验 -->
```
