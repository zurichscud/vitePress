# 唤起应用

## 基本流程

1. 浏览器发现不是 http(s)
2. 交给操作系统
3. 系统查找已注册该协议的应用
4. 有 → 打开应用。 没有 → 无反应 / 提示

## 常用应用

- 微信

```html
<a href="weixin://">打开微信</a>
```

- QQ

```html
<a href="mqq://">打开 QQ</a>
```

- 支付宝

```html
<a href="alipays://">打开支付宝</a>
```

- 电话 / 短信

```html
<a href="tel:13800138000">打电话</a>
<a href="sms:13800138000">发短信</a>
```
