# HTML字符实体

## 为什么需要字符实体

字符实体就是**用一段固定写法，来表示那些在 HTML 里“有特殊含义”或“键盘不好直接打”的字符**。

HTML多个连续的空格会被当做一个空格，我们使用HTML字符实体就可以解决这个问题：

```html
1&nbsp;&nbsp;&nbsp;&nbsp;22
```

## 常见字符实体

| 显示 | 实体 | 说明   |
| ---- | ---- | ------ |
| `<`  | `&lt;`  | 小于号 |
| `>`  | `&gt;`  | 大于号 |
| `&`  | `&amp;`  | 与号   |
| `"`  | `&quot;`  | 双引号 |
| `'`  | `&apos;`  | 单引号 |
|  | `&nbsp;` | 空格 |

可以发现字符实体的格式是

```html
& + 单词 + ;
```
