# 列表

## 有序列表

```html
<ol>
  <li>注册账号</li>
  <li>填写信息</li>
  <li>提交审核</li>
</ol>

```



## 无序列表

```html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橙子</li>
</ul>
```



## 定义列表

| 标签 | 含义     | 场景       |
| ---- | -------- | ---------- |
| `dl` | 定义列表 | 说明、解释 |
| `dt` | 定义标题 | 名词       |
| `dd` | 定义描述 | 解释       |

```html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言1</dd>
  <dd>超文本标记语言2</dd>

  <dt>CSS</dt>
  <dd>层叠样式表1</dd>
  <dd>层叠样式表2</dd>
</dl>

```

## 列表嵌套

`li`标签中可以再嵌套列表：

```html
<ul>
  <li>前端
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>后端</li>
</ul>

```
