# 列表样式

## display

| 标签 | 默认 display | 是否有 marker |
| ---- | ------------ | ------------- |
| `ul` | block        | ❌             |
| `ol` | block        | ❌             |
| `li` | list-item    | ✅             |
| `dt` | block        | ❌             |
| `dd` | block        | ❌             |

### display:list-item

这是一个**特殊值**，本质接近 `block`：

- 有块级行为（会换行）

- 额外渲染一个 `::marker`

`::marker` 就是那个小圆点 / 数字

```css
li::marker {
  color: red;
  font-size: 20px;
}

```
