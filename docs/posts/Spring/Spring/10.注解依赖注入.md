# 注解依赖注入

## @Value

`@Value` 是 Spring 提供的一个属性注入注解。使用`@Value`注入，可以不提供set方法

### 注入常量值

```java
@Value("hello world")
private String str;
```

### 注入配置文件的值

- 单个值

```java
@Value("${spring.datasource.url}")
private String url;
```

- 数组

```java
@Value("${app.names}")
private String[] names;
```

```yaml
app:
  names: a,b,c
```

### 在构造器上使用

```java
@Component
public class UserService {

    private final String name;
    private final int age;

    public UserService(
            @Value("${user.name}") String name,
            @Value("${user.age}") int age) {
        this.name = name;
        this.age = age;
    }
}
```

### 设置默认值

```java
@Value("${属性名:默认值}")
private String name;
```

```java
@Value("${server.port:8080}")
private int port;
```



## @Autowired

`@Autowired` 是 Spring 中用于**自动依赖注入（DI）**的核心注解。

`@Autowired`默认按 **类型（Type）匹配**，如果容器中有多个 `UserService`会报错。我们可以配合`@Qualifier`按Name匹配

###  属性注入

属性注入不需要提供set方法

```java
@Service
public class OrderService {

    @Autowired
    private UserService userService;

}
```

### 构造函数注入

```java
@Service
public class OrderService {

    private final UserService userService;

    @Autowired
    public OrderService(UserService userService) {
        this.userService = userService;
    }
}
```

### Set注入

```java
@Service
public class OrderService {

    private UserService userService;

    @Autowired
    public void setUserService(UserService userService) {
        this.userService = userService;
    }
}
```

## @Qualifier

`@Qualifier`配合`Autowired`可以按Name匹配

```java
@Autowired
@Qualifier("userServiceImplForMySql")
private UserService userService;
```

## @Resource

`@Resource` 是 Java 提供的一个依赖注入注解，来自 **Jakarta EE（原 Java EE）**，在 Spring 中同样被广泛支持。在项目中推荐使用`@Resource`代替`@Autowired`

### 默认按名称注入

```java
@Resource
private UserService userService;
```

Spring 会去找 **Bean 名叫 `userService` 的对象**

### 按类型注入

找不到名称 则按类型注入，如果类型存在唯一实现对象，则注入成功

### 显式指定名称注入

```java
@Resource(name = "userServiceImpl")
private UserService userService;
```
