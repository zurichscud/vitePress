# 构造注入

Spring直接调用构造函数对bean中的相关属性进行依赖注入

## 调用无参构造

如果bean没有写：

```xml
<bean id="orderService" class="com.example.OrderService"/>
```

Spring将调用无参构造创建对象，并使用set进行依赖注入

## 按顺序匹配（默认）

```java
public class OrderService {

    private UserRepository userRepository;
    private OrderRepository orderRepository;

    public OrderService(UserRepository userRepository,
                        OrderRepository orderRepository) {
        this.userRepository = userRepository;
        this.orderRepository = orderRepository;
    }
}
```

```xml
<bean id="userRepository" class="com.example.UserRepository"/>

<bean id="orderService" class="com.example.OrderService">
    <constructor-arg ref="userRepository"/>
    <constructor-arg ref="orderRepository"/>
</bean>
```

- 第一个 参数：userRepository
- 第二个参数：orderRepository

## 按索引注入

推荐用于多个参数，`index="0"`表示构造方法的第一个参数

```xml
<bean id="orderService" class="com.example.OrderService">
    <constructor-arg index="0" ref="userRepository"/>
</bean>
```

## 按形参名称注入

```xml
<bean id="orderService" class="com.example.OrderService">
    <constructor-arg name="userRepository" ref="userRepository"/>
    <constructor-arg name="paymentService" ref="paymentService"/>
</bean>
```

## 注入基本数据类型

```java
public class ConfigService {

    private String appName;
    private int timeout;

    public ConfigService(String appName, int timeout) {
        this.appName = appName;
        this.timeout = timeout;
    }
}
```

```xml
<bean id="configService" class="com.example.ConfigService">
    <constructor-arg value="MyApp"/>
    <constructor-arg value="30"/>
</bean>
```
