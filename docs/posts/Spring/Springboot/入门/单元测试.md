# 单元测试

## 依赖

Spring Boot 默认集成了测试框架

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-test</artifactId>
    <scope>test</scope>
</dependency>
```



## 基础用法

`@SpringBootTest`会启动整个Spring Boot容器

```java
@SpringBootTest
class UserServiceTest {

    @Autowired
    private UserService userService;

    @Test
    void testGetUser() {
        User user = userService.getById(1L);
        System.out.println(user);
    }
}
```

## 事务回滚

```java
@SpringBootTest
@Transactional
class UserTest {
}
```

每个测试执行后自动回滚，不污染数据库
