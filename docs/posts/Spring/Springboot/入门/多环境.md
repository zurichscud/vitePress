# 多环境

在实际开发环境中，Springboot需要在不同的环境中运行，例如开发环境，测试环境，生产环境。spring提供了配置项，它可以让我们快速切换环境。

多环境配置文件通常包括：

```
application-dev-yml
application-test.yml
application-prod.yml
```

在`application.properties`中使用以下配置激活相应环境

```properties
spring.profiles.active=dev #也可以是test、prod
```

我们也可以在运行时指定配置环境

```sh
java -jar springboot-demo.jar --spring.profiles.active=test
```
