# 配置绑定到Bean

## @ConfigurationProperties

`@ConfigurationProperties` 是 **Spring / Spring Boot 用来把配置文件中的属性批量绑定到 Bean上的注解**。它常用于读取 `application.yml` 或 `application.properties` 中的一组配置。

::: warning

- 配置文件需要有getter和setter
- `@ConfigurationProperties` 本身 **不会创建 Bean**，因此需要添加`@Component`或`@Configuration`

:::

### 绑定到嵌套Bean

::: code-group



```yml [application.yaml]
app:
  name: my-app
  security:
    enabled: true
    roles:
      - admin
      - user
```

```java [SecurityProperties]
@Data
public class SecurityProperties {
    private boolean enabled;
    private List<String> roles;
}
```

```java [AppProperties]
import org.springframework.boot.context.properties.ConfigurationProperties;
import org.springframework.stereotype.Component;
import lombok.Data;

@Data
@Component
@ConfigurationProperties(prefix = "app")
public class AppProperties {

    private String name;

    // 嵌套对象
    private SecurityProperties security;
}
```

:::



### 绑定到Map

::: code-group

```yaml [application.yml]
app:
  configs:
    key1: value1
    key2: value2
```

```java [AppProperties.java]
@Data
@Component
@ConfigurationProperties(prefix = "app")
public class AppProperties {

    private Map<String, String> configs;
}
```





:::



### 绑定到对象数组



::: code-group

```yaml [application.yml]
app:
  users:
    - name: 张三
      age: 18
    - name: 李四
      age: 20
```

```java [User.java]
@Data
public class User {
    private String name;
    private Integer age;
}
```

```java [AppProperties.java]
@Data
@Component
@ConfigurationProperties(prefix = "app")
public class AppProperties {

    private List<User> users;
}
```



:::



## 配置类没有@Component

配置绑定到Bean的前提是配置类必须被Spring管理。但是有些第三方配置类并没有被Spring容器管理，我们也不可能在他人的源代码上添加`@Component`。我们可以使用`@EnableConfigurationProperties`解决这个问题。

例如：

```java
/*源码无法修改*/
@ConfigurationProperties(prefix = "user")
public class UserProperties {

    private String name;
    private Integer age;

}
```

`@EnableConfigurationProperties`可以帮助我们将指定的Bean添加到Spring容器管理：

```java
@EnableConfigurationProperties(UserProperties.class)
public class RuoYiApplication{}
```

## 配置类没有@ConfigurationProperties

使用 `@Bean + @ConfigurationProperties`，把第三方类注册成 Bean 时顺便绑定配置

::: code-group

```java [ThirdService.java]
public class ThirdService {
    private String url;
    private Integer timeout;

    // getter / setter
}
```

```yaml [application.yml]
third:
  url: http://api.xxx.com
  timeout: 5000
```

```java [ThirdConfig.java]
@Configuration
public class ThirdConfig {

    @Bean
    @ConfigurationProperties(prefix = "third")
    public ThirdService thirdService() {
        return new ThirdService();
    }
}
```



:::



## 指定配置文件来源

```java
@Configuration
@PropertySource("classpath:custom.properties")
public class CustomConfig {
}
```

- 不支持 `.yml`配置文件

- 只支持 `.properties`配置文件
