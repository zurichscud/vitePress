# Scheduler

## 启动 / 停止 Scheduler

| 方法              | 说明                                                         |
| ----------------- | ------------------------------------------------------------ |
| `void start()`    | 启动 Scheduler（调度器开始执行任务）                         |
| `void standby()`  | 暂停 Scheduler，但保留已注册任务（可恢复）                   |
| `void shutdown()` | 关闭 Scheduler，可选择 `true/false` 是否等待正在执行的任务完成 |

```java
scheduler.start();           // 启动
scheduler.standby();         // 暂停
scheduler.shutdown(true);    // 关闭，等待任务完成
```

## Job 管理

| 方法                                                   | 说明                                                  |
| ------------------------------------------------------ | ----------------------------------------------------- |
| `addJob(JobDetail jobDetail, boolean replace)`         | 注册 JobDetail，如果 `replace=true`，已有任务会被覆盖 |
| `boolean deleteJob(JobKey jobKey)`                     | 删除指定 Job                                          |
| `boolean checkExists(JobKey jobKey)`                   | 判断 Job 是否存在                                     |
| `List<String> getJobGroupNames()`                      | 获取所有 Job 分组名称                                 |
| `Set<JobKey> getJobKeys(GroupMatcher<JobKey> matcher)` | 获取指定分组的 JobKeys                                |

## Trigger管理

| 方法                                                         | 说明                             |
| ------------------------------------------------------------ | -------------------------------- |
| `scheduleJob(JobDetail jobDetail, Trigger trigger)`          | 注册 Job + Trigger（一次性创建） |
| `scheduleJob(Trigger trigger)`                               | 给已存在 Job 添加 Trigger        |
| `rescheduleJob(TriggerKey triggerKey, Trigger newTrigger)`   | 更新 Trigger（如修改 Cron）      |
| `unscheduleJob(TriggerKey triggerKey)`                       | 移除 Trigger（Job 可能仍存在）   |
| `boolean checkExists(TriggerKey triggerKey)`                 | 判断 Trigger 是否存在            |
| `Set<TriggerKey> getTriggerKeys(GroupMatcher<TriggerKey> matcher)` | 获取 TriggerKey 集合             |
