# Listener

在 Quartz 框架中，**Listener（监听器）** 相当于给调度流程安装了“全方位的监控摄像头”。

它允许你在任务执行的前后、触发器触发的瞬间、或者调度器发生错误时，插入自己的逻辑（如：打印审计日志、统计执行时长、发送报警通知等）。

| **监听器类型**        | **监控对象**           | **核心作用**                                                 |
| --------------------- | ---------------------- | ------------------------------------------------------------ |
| **JobListener**       | **Job** (任务)         | 监听任务的：准备执行、执行完成、被否决（被 TriggerListener 拦下）。 |
| **TriggerListener**   | **Trigger** (触发器)   | 监听触发器的：触发、错过（Misfire）、完成。                  |
| **SchedulerListener** | **Scheduler** (调度器) | 监听调度器的：启动、关闭、Job 的添加/删除、严重的系统错误。  |

## Job Listener

### JobListener接口

你需要实现 `org.quartz.JobListener` 接口。为了方便，通常继承 `JobListenerSupport` 类，这样你只需要重写你感兴趣的方法。

```java
public interface JobListener {
    String getName(); // listener 名称

    void jobToBeExecuted(JobExecutionContext context); // 执行前
    void jobExecutionVetoed(JobExecutionContext context); // 被阻止执行
    void jobWasExecuted(JobExecutionContext context, JobExecutionException jobException); // 执行后
}
```



```java
import org.quartz.*;
import org.quartz.listeners.JobListenerSupport;

public class MyJobListener extends JobListenerSupport {

    @Override
    public String getName() {
        return "myJobListener";
    }

    @Override
    public void jobToBeExecuted(JobExecutionContext context) {
        System.out.println("Job will execute: " + context.getJobDetail().getKey());
    }

    @Override
    public void jobExecutionVetoed(JobExecutionContext context) {
        System.out.println("Job execution vetoed: " + context.getJobDetail().getKey());
    }

    @Override
    public void jobWasExecuted(JobExecutionContext context, JobExecutionException jobException) {
        System.out.println("Job was executed: " + context.getJobDetail().getKey());
        if (jobException != null) {
            System.out.println("Exception: " + jobException.getMessage());
        }
    }
}
```



### 注册JobListener

定义好监听器后，你需要告诉 **Scheduler** 去使用它。你可以设置为“全局监听”或者“针对特定任务监听”。

- A. 全局监听 (监控所有任务)

```java
scheduler.getListenerManager().addJobListener(new MyJobListener(), EverythingMatcher.allJobs());
```



- B. 局部监听 (只监控特定组的任务)

```java
scheduler.getListenerManager().addJobListener(new MyJobListener(), GroupMatcher.jobGroupEquals("MemberGroup"));
```



- C. 监听具体任务

```java
// 1. 创建你的监听器
JobListener myListener = new MyJobListener();

// 2. 定义你想监听的 Job 的 Key (必须与创建 JobDetail 时的一致)
JobKey targetJobKey = new JobKey("expireReminderJob", "memberGroup");

// 3. 注册监听器并绑定匹配器
scheduler.getListenerManager().addJobListener(
    myListener, 
    KeyMatcher.keyEquals(targetJobKey) // 只有这个 Key 匹配时才触发
);
```





## Trigger Listener

### TriggerListener接口

```java
public interface TriggerListener {
    String getName();  // 必须实现，返回 listener 名称

    void triggerFired(Trigger trigger, JobExecutionContext context); // 触发器触发时
    boolean vetoJobExecution(Trigger trigger, JobExecutionContext context); // 是否阻止 job 执行
    void triggerMisfired(Trigger trigger); // 触发器错过执行
    void triggerComplete(Trigger trigger, JobExecutionContext context, Trigger.CompletedExecutionInstruction triggerInstructionCode); // 执行完成
}
```



```java
import org.quartz.*;
import org.quartz.listeners.TriggerListenerSupport;

public class MyTriggerListener extends TriggerListenerSupport {

    @Override
    public String getName() {
        return "myTriggerListener";
    }

    @Override
    public void triggerFired(Trigger trigger, JobExecutionContext context) {
        System.out.println("Trigger fired: " + trigger.getKey());
    }

    @Override
    public boolean vetoJobExecution(Trigger trigger, JobExecutionContext context) {
        System.out.println("Veto check for: " + trigger.getKey());
        return false; // 返回true将阻止Job执行
    }

    @Override
    public void triggerMisfired(Trigger trigger) {
        System.out.println("Trigger misfired: " + trigger.getKey());
    }

    @Override
    public void triggerComplete(Trigger trigger, JobExecutionContext context, Trigger.CompletedExecutionInstruction triggerInstructionCode) {
        System.out.println("Trigger complete: " + trigger.getKey());
    }
}
```



### 注册 TriggerListener

```java
Scheduler scheduler = StdSchedulerFactory.getDefaultScheduler();

// 创建listener
MyTriggerListener listener = new MyTriggerListener();

// 注册到scheduler
scheduler.getListenerManager().addTriggerListener(listener);

// 也可以使用Matcher指定监听特定trigger
// scheduler.getListenerManager().addTriggerListener(listener, KeyMatcher.keyEquals(TriggerKey.triggerKey("trigger1")));
```





## Scheduler Listener

Quartz 的 **Scheduler Listener** 是 Quartz 监听器体系中最“顶层”的监听器，它不是针对单个 Job 或 Trigger，而是监听整个 **Scheduler 的生命周期和整体事件**。可以理解为管理层级别的监听器。

**调度器生命周期事件**：

- `schedulerStarting()`：Scheduler 即将启动
- `schedulerStarted()`：Scheduler 已启动
- `schedulerShuttingdown()`：Scheduler 正在关闭
- `schedulerShutdown()`：Scheduler 已关闭
