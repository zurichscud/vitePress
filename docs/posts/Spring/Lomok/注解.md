# 注解

## EqualsAndHashCode

> **不仅比较当前类字段，还会比较父类字段**

```java
class BaseEntity {
    private Long id;
}
```

```java
@Data
class User extends BaseEntity {
    private String name;
}
```

如果没有`@EqualsAndHashCode`，将会完全忽略父类字段（id）

```java
User u1 = new User();
u1.setId(1L);
u1.setName("A");

User u2 = new User();
u2.setId(2L);
u2.setName("A");
```

```java
u1.equals(u2)  // 👉 true ❗❗❗
```

该注解的本质：

```java
@Override
public boolean equals(Object o) {
    if (!super.equals(o)) return false; // 👈 先比较父类字段

    User other = (User) o;
    return Objects.equals(this.name, other.name);//再比较子类字段
}
```

## Accessors

让Bean支持链式调用

```java
@Data
@Accessors(chain = true)
public class User {
    private String name;
    private Integer age;
}
```

```java
User user = new User()
        .setName("张三")
        .setAge(18);
```
