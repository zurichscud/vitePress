# 全局配置文件

`lombok.config` 是 Project Lombok 提供的一个**项目级配置文件**，用于控制 Lombok 在编译阶段的行为（比如注解生成代码的方式、风格约束等）。

## 配置文件

```[lombok.config]
config.stopBubbling = true
lombok.tostring.callsuper = call
lombok.equalsandhashcode.callsuper = call
lombok.accessors.chain=true
```

## 如何生效

只要 Lombok 参与编译，它就会：

1. 找到当前 Java 文件所在目录
2. 向上查找 `lombok.config`
3. 合并配置并应用
4. 决定生成什么代码（getter / equals / builder 等）



Lombok 会**从当前类所在目录向上查找** `lombok.config`，直到：

- 找到配置文件
- 或遇到 `config.stopBubbling = true`

## 优先级

注解参数  >  lombok.config  >  Lombok 默认值
