# 安装与配置

## 安装

进入https://maven.apache.org/download.cgi 下载已编译的包

- MacOS 下载apache-maven-3.9.12-bin.tar.gz

- WIndows下载apache-maven-3.9.12-bin.zip

解压至本地，至此安装结束

## 配置镜像

进入安装目录的`conf/settings.xml`，配置阿里云镜像：

```xml
<mirror>
  <id>aliyunmaven</id>
  <mirrorOf>central</mirrorOf>
  <name>阿里云公共仓库</name>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

## 配置环境变量

```sh
# MAVEN
export MAVEN_HOME=~/Developer/apache-maven-3.9.12
export PATH=$PATH:$MAVEN_HOME/bin
```

验证是否配置成功

```sh
mvn -v
```
