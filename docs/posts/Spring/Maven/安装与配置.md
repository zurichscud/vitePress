# 安装与配置

## 安装

进入https://maven.apache.org/download.cgi下载已编译的包

- MacOS 下载[apache-maven-3.9.12-bin.tar.gz](https://dlcdn.apache.org/maven/maven-3/3.9.12/binaries/apache-maven-3.9.12-bin.tar.gz)

- WIndows下载[apache-maven-3.9.12-bin.zip](https://dlcdn.apache.org/maven/maven-3/3.9.12/binaries/apache-maven-3.9.12-bin.zip)

解压至本地，至此安装结束

## 配置镜像

进入安装目录的`conf/settings.xml`，配置阿里云镜像：

```xml
<mirror>
  <id>aliyunmaven</id>
  <mirrorOf>central</mirrorOf>
  <name>阿里云公共仓库</name>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```
