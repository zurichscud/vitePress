# JSR-303



## 判空

### NotNull

不能为 null

### NotEmpty

不能为 null + 空字符串/**空集合**

### NotBlank

不能为 null + 空字符串 + 纯空格

## 字符串校验



```java
@Size(min = 3, max = 10)
private String name;
```



```java
@Pattern(regexp = "^[a-zA-Z0-9]+$")
private String code;
```



## 数值校验

```java
@Min(1)
@Max(100)
private Integer age;
```



```java
@DecimalMin("0.1")
@DecimalMax("10.0")
private BigDecimal price;
```



## 日期校验

```java

@Future      // 必须是未来时间
@Past        // 必须是过去时间
```

## Controller 开启校验

```java
@PostMapping("/add")
public String add(@Valid @RequestBody UserDTO dto) {
    return "ok";
}
```

`@Valid`：触发校验





## 统一处理

```java
@RestControllerAdvice
public class GlobalExceptionHandler {

    @ExceptionHandler(MethodArgumentNotValidException.class)
    public String handle(MethodArgumentNotValidException e) {
        return e.getBindingResult().getFieldError().getDefaultMessage();
    }
}
```

## 自定义提示信息

```java
@NotNull(message = "用户名不能为空")
private String username;
```



## @Validated

```java
@RestController
@RequestMapping("/product/category")
@Validated
public class AppCategoryController {}
```

| 对比         | @Valid                 | @Validated |
| ------------ | ---------------------- | ---------- |
| 来源         | JSR规范                | Spring     |
| 是否支持分组 | ❌ 不支持               | ✅ 支持     |
| 方法参数校验 | ❌ 不稳定（仅部分场景） | ✅ 支持     |
| 嵌套对象校验 | ✅ 支持                 | ✅ 支持     |

```java
@PostMapping("/test")
public void test(@NotNull String name) {
}
```
