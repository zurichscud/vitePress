# 运维字段

```sql
ALTER TABLE ib_group
ADD COLUMN create_by VARCHAR(64) NULL COMMENT '创建人',
ADD COLUMN create_time DATETIME NULL COMMENT '创建时间',
ADD COLUMN update_by VARCHAR(64) NULL COMMENT '更新人',
ADD COLUMN update_time DATETIME NULL COMMENT '更新时间',
ADD COLUMN is_deleted TINYINT(1) DEFAULT 0 COMMENT '是否删除（0：未删除，1：已删除）';
```
