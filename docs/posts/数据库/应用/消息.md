# 消息

| 字段        | 说明     |
| ----------- | -------- |
| id          | 消息ID   |
| user_id     | 用户     |
| title       | 标题     |
| content     | 内容     |
| is_read     | 是否已读 |
| type        | 消息类型 |
| create_time | 创建时间 |

```http
# 获得未读数量
GET /message/unreadCount
# 查看消息列表
GET /message/list
# 标记已读
POST /message/read
```
